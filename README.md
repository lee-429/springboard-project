# SpringBoard Full Stack

Spring Boot + React 기반 JWT 인증 게시판 서비스 프로젝트

## 📌 Project Overview

Spring Boot REST API와 React를 활용하여 구현한 Full Stack 게시판 서비스 프로젝트입니다.

백엔드 API 서버 구축부터 React 기반 사용자 인터페이스 구현,
JWT 인증 처리 및 프론트엔드-백엔드 통신까지 실제 서비스 구조를 목표로 개발했습니다.

---

## 🛠 Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- JWT
- MySQL
- Gradle

### Frontend
- React
- Vite
- Axios

---

## ✨ Features

### Authentication
- 회원가입 / 로그인
- JWT Access Token / Refresh Token 인증
- Axios Interceptor 기반 인증 처리
- 로그인 상태 관리

### Board
- 게시글 CRUD
- 게시글 검색
- 페이징 처리
- 파일 업로드

### Comment
- 댓글 CRUD
- 작성자 권한 검증

### Frontend
- React 컴포넌트 기반 UI 구성
- 게시글 목록 / 상세 / 작성 화면 구현
- 로그인 사용자 UI 처리
- API 연동
